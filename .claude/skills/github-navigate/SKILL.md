# GitHub Documentation Navigation

Navigate official GitHub documentation with an always-current index.

## When to Use

Use this skill when users ask about:
- GitHub Actions (workflows, CI/CD, actions syntax)
- GitHub Copilot (setup, usage, features)
- GitHub REST API and GraphQL API
- Repository management (branches, commits, PRs)
- GitHub authentication (SSH, tokens, SSO)
- GitHub security features (code scanning, Dependabot, secret scanning)
- GitHub Codespaces
- GitHub Pages
- GitHub Issues and Projects
- Organization and enterprise administration
- GitHub CLI
- GitHub Apps and integrations

## Index Location

The documentation index is at: `data/index.md`

## How to Navigate

1. Read `data/index.md` to find the relevant section
2. Follow links to topic summaries
3. Fetch specific documentation from the source repository when needed
